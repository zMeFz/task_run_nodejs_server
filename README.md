# Tasks running server (TRS)

### Installation

TRS requires [Node.js](https://nodejs.org/) v6+ to run.

#### TRS installation

You need clone the repository.

Then navigate into the folder, and run:

```sh
$ npm i
```

#### TRS start

```sh
$ node index.js
```
Then open http://localhost:3000/
